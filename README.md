# 🍳 AI Kitchen Companion

An AI-powered mobile application that helps users discover recipes using ingredients already available in their pantry, reduce food waste, and organize kitchen planning.

## 📱 Features

### 🥫 Pantry Management

* View available pantry ingredients
* Select ingredients currently in stock
* Track items that may expire soon

### 🍳 Smart Recipe Suggestions

* Generate recipes based on selected ingredients
* Match recipes according to available pantry items
* Highlight the best recipe match

### ❤️ Save Favorite Recipes

* Save recipes for future use
* Remove saved recipes anytime
* Persistent storage using AsyncStorage

### 🤖 AI Assistant

* View saved recipes
* Central hub for future AI-powered kitchen features

### 🎨 Modern Mobile UI

* Clean and intuitive interface
* Bottom tab navigation
* Responsive card-based design

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/janhavipatkar854/AI-Kitchen-Companion.git
```

### Navigate to Project

```bash
cd AI-Kitchen-Companion
```

### Install Dependencies

```bash
npm install
```

### Start Expo

```bash
npx expo start
```

AI Kitchen Companion helps users:
